# Personal CV Webpage

This project is a personal CV webpage created as part of a laboratory assignment. The webpage showcases my skills, education, and interests in the field of IT and electronics.

## Project Structure

- **index.html**: The main HTML document for the personal CV webpage. It includes the HTML structure, headings, paragraphs, links, images, and lists as specified in the requirements.
- **css/styles.css**: This file contains the CSS styles for the HTML elements in the index.html file. It is used to style the layout, fonts, colors, and other visual aspects of the webpage.
- **images/**: This directory is intended to store image files used in the index.html file. It should contain images that reflect my dream job or other relevant visuals.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the project folder in your code editor.
3. Use the Live Server extension to view the index.html file in your web browser.
4. Ensure that all images are placed in the `images` directory and linked correctly in the index.html file.
